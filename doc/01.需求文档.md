# HTML 交互式教学网站 - 需求文档

## 项目定位

面向 HTML 零基础新人的中文交互式教学文档网站。

## 技术栈

- **核心框架**：React（生态不限）
- **类型**：纯前端，无后端
- **UI**：简约风格，使用 UI 组件库
- **部署**：Nginx / GitHub Pages（静态站点）

## 功能模块

### 一、文档区

- 系统讲解 HTML 基础元素，通俗易懂
- 文档类网站风格，无用户系统

### 二、内嵌演示区（穿插在文档中）

- 每个知识点配合可运行的代码示例
- 阅读过程中直接看到渲染效果

### 三、交互实验区（章节结尾）

- 代码编辑器 + 实时预览
- 用户自由编写，即时看到结果
- 类似 CodePen / JSFiddle 的简易版

## 内容规划

- 第一阶段：HTML 基础元素
- 后续逐步补全表单、表格、语义化、多媒体等

## 差异化

现有市场产品大多侧重文档或互动练习单一方向，本项目将文档学习、实例演示、动手实践整合在同一页面内，形成「学 → 看 → 练」闭环。

## 约束

- 暂不考虑移动端适配
- 先出一版基础版本
