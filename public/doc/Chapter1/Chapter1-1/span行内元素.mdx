# HTML `span` 元素

`span` 是一个没有额外语义的行内容器，适合对句子里的局部文本做样式区分。  
它不会像 `div` 那样单独占一行，更适合放在段落、标题、按钮文字内部。

## 一个常见示例

:::code{html="<p>&#10;  当前状态：<span class='status'>运行中</span>，最近一次部署已完成。&#10;</p>" css="body{&#10;  font-family:'Segoe UI',sans-serif;&#10;  padding:24px;&#10;  background:#f8fafc;&#10;}&#10;p{&#10;  max-width:680px;&#10;  margin:0 auto;&#10;  font-size:18px;&#10;  line-height:1.8;&#10;  color:#334155;&#10;}&#10;.status{&#10;  padding:2px 10px;&#10;  border-radius:999px;&#10;  background:#dcfce7;&#10;  color:#166534;&#10;  font-weight:700;&#10;}" js=""}
:::

## 适用场景

- 高亮关键词
- 插入状态徽标
- 局部颜色、字重、装饰变化

## 不适用场景

如果你需要包裹的是一整块区域，或者要做结构分组，那应该改用 `div`、`section` 之类的块级标签。
