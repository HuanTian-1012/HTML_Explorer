# HTML `button` 元素

`button` 表示一个可触发动作的控件。  
它适合提交、确认、切换状态、打开弹窗，而不是承担页面跳转职责。

## 一个交互示例

:::code{html="<div class='panel'>&#10;  <p id='text'>当前模式：普通</p>&#10;  <button id='toggle'>切换模式</button>&#10;</div>" css="body{&#10;  font-family:'Segoe UI',sans-serif;&#10;  padding:24px;&#10;  background:#f8fafc;&#10;}&#10;.panel{&#10;  max-width:420px;&#10;  margin:0 auto;&#10;  padding:24px;&#10;  border-radius:18px;&#10;  background:#fff;&#10;  box-shadow:0 18px 40px rgba(15,23,42,0.08);&#10;  text-align:center;&#10;}&#10;p{&#10;  margin:0 0 16px;&#10;  color:#334155;&#10;  font-size:18px;&#10;}&#10;button{&#10;  border:none;&#10;  border-radius:999px;&#10;  padding:12px 20px;&#10;  background:#0f172a;&#10;  color:#fff;&#10;  font-size:16px;&#10;  cursor:pointer;&#10;}&#10;button:hover{&#10;  background:#1e293b;&#10;}" js="const text = document.getElementById('text');&#10;const button = document.getElementById('toggle');&#10;let active = false;&#10;&#10;button.addEventListener('click', () => {&#10;  active = !active;&#10;  text.textContent = `当前模式：${active ? '高亮' : '普通'}`;&#10;});"}
:::

## 使用建议

- 触发动作就用 `button`
- 需要提交表单时设置正确的 `type`
- 只做跳转时优先用 `a`
