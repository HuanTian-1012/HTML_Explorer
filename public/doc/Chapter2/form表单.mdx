# HTML `form` 元素

`form` 用来承载一组需要提交的数据。  
输入框、选择框、提交按钮只有放在表单语义里，浏览器和辅助技术才能更完整地理解交互目标。

## 一个示例

:::code{html="<form class='form'>&#10;  <label>&#10;    邮箱&#10;    <input type='email' placeholder='name@example.com'>&#10;  </label>&#10;  <label>&#10;    昵称&#10;    <input type='text' placeholder='请输入昵称'>&#10;  </label>&#10;  <button type='submit'>提交</button>&#10;</form>" css="body{&#10;  font-family:'Segoe UI',sans-serif;&#10;  padding:24px;&#10;  background:#f0fdf4;&#10;}&#10;.form{&#10;  max-width:420px;&#10;  margin:0 auto;&#10;  padding:24px;&#10;  border-radius:18px;&#10;  background:#fff;&#10;  box-shadow:0 18px 40px rgba(22,101,52,0.08);&#10;  display:grid;&#10;  gap:16px;&#10;}&#10;label{&#10;  display:grid;&#10;  gap:8px;&#10;  color:#166534;&#10;  font-weight:600;&#10;}&#10;input{&#10;  padding:12px 14px;&#10;  border:1px solid #bbf7d0;&#10;  border-radius:12px;&#10;  font-size:15px;&#10;}&#10;button{&#10;  border:none;&#10;  border-radius:12px;&#10;  padding:12px 16px;&#10;  background:#166534;&#10;  color:#fff;&#10;  font-size:16px;&#10;  cursor:pointer;&#10;}" js="document.querySelector('.form')?.addEventListener('submit', (event) => {&#10;  event.preventDefault();&#10;  alert('表单已提交');&#10;});"}
:::

## 结构重点

1. 表单负责组织提交范围。
2. `label` 负责建立可访问的说明关系。
3. `button type='submit'` 负责触发提交。

## 总结

表单不是单个输入框的集合，而是一段完整的数据收集流程。
