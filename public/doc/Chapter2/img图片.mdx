# HTML `img` 元素

`img` 用来在页面中嵌入图片资源。  
除了展示图像本身，更重要的是给出清晰的 `alt` 文本，让无法看到图片的用户也能理解内容。

:::code{html="<figure><img src='https://images.unsplash.com/photo-1500530855697-b586d89ba3ee?auto=format&fit=crop&w=900&q=80' alt='湖边山脉与晨雾风景'><figcaption>清晨的山谷景象。</figcaption></figure>" css="body{font-family:'Segoe UI',sans-serif;padding:24px;background:#e0f2fe;} figure{max-width:760px;margin:0 auto;padding:20px;border-radius:20px;background:#fff;box-shadow:0 18px 40px rgba(3,105,161,0.12);} img{display:block;width:100%;border-radius:14px;object-fit:cover;} figcaption{margin-top:12px;color:#475569;line-height:1.7;}" js=""}
:::

## 使用建议

- 装饰图可用空 `alt`
- 内容图要写清楚 `alt`
- 控制尺寸时尽量避免图片拉伸变形
