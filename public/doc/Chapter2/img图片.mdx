# HTML `img` 元素

`img` 用来在页面中嵌入图片资源。  
除了展示图像本身，更重要的是给出清晰的 `alt` 文本，让无法看到图片的用户也能理解内容。

## 一个示例

:::code{html="<figure>&#10;  <img src='https://images.unsplash.com/photo-1500530855697-b586d89ba3ee?auto=format&fit=crop&w=900&q=80' alt='湖边山脉与晨雾风景'>&#10;  <figcaption>清晨的山谷景象。</figcaption>&#10;</figure>" css="body{&#10;  font-family:'Segoe UI',sans-serif;&#10;  padding:24px;&#10;  background:#e0f2fe;&#10;}&#10;figure{&#10;  max-width:760px;&#10;  margin:0 auto;&#10;  padding:20px;&#10;  border-radius:20px;&#10;  background:#fff;&#10;  box-shadow:0 18px 40px rgba(3,105,161,0.12);&#10;}&#10;img{&#10;  display:block;&#10;  width:100%;&#10;  border-radius:14px;&#10;  object-fit:cover;&#10;}&#10;figcaption{&#10;  margin-top:12px;&#10;  color:#475569;&#10;  line-height:1.7;&#10;}" js=""}
:::

## 使用建议

- 装饰图可用空 `alt`
- 内容图要写清楚 `alt`
- 控制尺寸时尽量避免图片拉伸变形
